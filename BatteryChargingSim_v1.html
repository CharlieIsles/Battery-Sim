// Battery Charging Simulator

This tool helps you understand how far a set of drone batteries will get you through a working day, given how they're charged and used.

Each battery starts the day fully charged. The simulation tracks every battery minute by minute — when it's in the drone, sitting idle, or on the charger — and tells you how many flights you can complete before things run dry.

Use the controls on the left to set up your equipment. The timeline shows each battery as a row: green while flying, red while charging, grey while idle. The height of each bar shows the charge level. A ☠️ marks the moment a flight couldn't be completed due to insufficient charge.

The charging window control lets you restrict when batteries can be charged — drag the handles to set the available window.
